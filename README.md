# Borderless Modding
  
.NET Framework製ボーダーレスウィンドウ化ソフトウェアです
  
◆◇◆使用方法◆◇◆
・ソフトウェアを起動し、コンボボックスよりボーダーレス化したいソフトウェアを選択します
・ボーダーレス化をクリックして　おわおわり

◆◇◆そのほか◆◇◆    
・非対応のソフトがあったなどありましたら報告していただければ可能であれば対応いたします
・起動には管理者権限が必要です
